المصطلح "API" يرمز إلى "Application Programming Interface"، وبالعربية يعني "واجهة برمجة التطبيقات". يتمثل مفهوم API في أنها تعرض مجموعة من الأوامر والوظائف والبرمجيات التي تمكن التطبيقات من التفاعل مع بعضها البعض. تسمح واجهة برمجة التطبيقات بالوصول إلى ميزات أو بيانات معينة أو خدمات من تطبيق أو نظام آخر بطرق محددة وموثوقة. تسهم API في تطوير التطبيقات وزيادة التكامل بينها.
